## Installing
```bash
npm install axios-response-logger --save
```

## Usage
Simply require / import the library as part of your bootstrapping (before your requests).
```es6
// es5
require('axios-response-logger');

// es6
import 'axios-response-logger';
```

## Running Tests
```bash
npm test
```
